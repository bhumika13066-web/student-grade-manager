# Student Grade Manager (GUI)

A lightweight Java desktop application built using the Swing framework. This application provides a simple graphical user interface (GUI) to input student names and grades, track them dynamically in memory, and generate an analytical report featuring individual grades, the class average, and the highest/lowest scores.

---

## Features

* **Interactive GUI:** Easy-to-use interface built entirely with Java Swing.
* **Dynamic Data Storage:** Uses Java `ArrayList` collections to store student records on the fly.
* **Instant Report Generation:** Calculates and displays:
  * A complete roster of entered students and their grades.
  * The class **Average Score**.
  * The **Highest Score** in the class.
  * The **Lowest Score** in the class.

---

## Prerequisites

Before running this project, ensure you have the following installed:
* **Java Development Kit (JDK):** Version 8 or higher.
* A command-line terminal or an IDE (like IntelliJ IDEA, Eclipse, NetBeans).

---

## How to Run

1. Clone the repository:
   ```bash
   git clone https://[github.com/bhumika13066-web/hotel-reservation-system](https://github.com/bhumika13066-web/student-grade-manager).git
   ```
2. Open the project folder in your terminal or IDE.
3. Compile the program:
   ```bash
   javac StudentGradeManagerGUI.java
   ```
4. Run the application:
   ```bash
   java StudentGradeManagerGUI
   ```
